This is a project walkthrough:

The folder contains 3 different script.
The tiktok_crawl.py is a module that will be responsible for Tiktok profiles scraping (Profile name, number of followers, number of followees,total number of likes, biography, profile image url, verified status).
The yt_crawl.py is a module that will be responsible for Youtube profiles scraping (Profile name, number of followers, number of followees,total number of likes, biography, profile image url, verified status).
The stream_test.py is the actual web application you can run it using the following command in the command line:

streamlit run stream_test.py

You also have to install the requirements specified in the requirements file and install mongodb locally then use the following localhost "mongodb://localhost:27017/" or modify it accordingly in the stream_test.py
